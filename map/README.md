# map
맵은 키와 값 형태로 데이터를 저장하는 자료구조 
언어에 따라 딕셔너리, 해쉬테이블, 해쉬맵 등으로 불린다.

값을 넣은 순서대로 값이 나오지 않는다.
맵은 크게 두가지로 나뉜다. 
- hash map(unordered map): 순서가 보장되지 않는다.
- sorted map: 순서가 보장된다. 

go에서는 hash map을 사용한다.


### 맵 순회하기 
```
for k, v := m {
    fmt.Println(k, v)
}
```

### 요소 삭제하기 
```
delete(m, key)
```

### 존재 여부
ok를 통해서 여부를 알 수 있다. 
```
v, ok = m[3]
```

### 맵의 원리 
해쉬 함수의 동작을 이용한다. 
해쉬 함수의 단방향성을 이용하여 
같은 입력이 들어오면 같은 값,
다른 입력이 들어오면 다른 값이 나오도록 한다. 

sin 함수를 해쉬함수로 사용할 수 있다. 
일반적으로 나머지 연산을 주로 사용한다.
